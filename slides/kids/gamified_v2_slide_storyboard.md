# Gamified v2 Slide Storyboard

## Purpose

v1の子供向けスライド本文は残す。
v2では、各スライドを「発見ゲーム型の画面」に作り直す。

このファイルは、画像生成後にPPTXまたは16:9 HTMLへ組み込むための対応表である。

## Common Slide Frame

- 16:9。
- 上部: 今日のミッション名。
- 中央: 画像または図解。
- 下部: 短い問い、またはワーク指示。
- 右下: 小さな進行マーカー。点数ではなく現在地。
- 親の説明はスライドに入れすぎず、台本に残す。

## Visual Roles

| Role | Use |
|---|---|
| Mission Board | 各日の1枚目 |
| Card Table | カードを選ぶ、並べる |
| Evidence Board | Before/Afterをつなぐ |
| Case File | 具体的な人を見る |
| Experiment Lab | 小さな実験を決める |
| Signal Radar | 欲しいサインを見る |
| Pitch Stage | 3分ピッチ |
| Quest Log | まとめ |

## Do Not

- 既存作品名をpromptやスライドに入れない。
- 既存キャラクターに似せない。
- バトル勝利、ランキング、レア度、点数で盛り上げない。
- 親を司令官やゲームマスターにしない。
