# 00_CODEX_START_HERE

Codexは、このファイルを最初に読むこと。

## 今回のミッション

中学2年生2人が、起業セミナーに参加する前に、4日間の家庭内ワークショップを通じて次を理解する教材一式を作る。

- 価値とは何か。
- 社会に、自分ができることで価値を提供する意味は何か。
- 困りごとや欲しい気持ちを解決すると仕事が生まれること。
- 価値が届いた対価としてお金を受け取ること。
- 起業セミナー本番が楽しそうだと思えること。

## 最重要ルール

このプロジェクトは、子供に「起業家になれ」と教えるものではない。

会社に入るとしても、自分で何かを始めるとしても役立つ、「誰かにとって価値あることを見つけ、形にし、小さく試す力」を育てる事前学習である。

## 実行順

1. `01_PROJECT_REQUIREMENTS.md` を読み、要件を理解する。
2. `02_SOURCE_OF_TRUTH.md` を読み、絶対にぶらさない表現を確認する。
3. `03_FOUR_DAY_CURRICULUM.md` を読み、4日間構成を固定する。
4. `04_DELIVERABLES_SPEC.md` を読み、成果物の範囲を確認する。
5. `05_ACCEPTANCE_CRITERIA.md` を読み、品質基準を確認する。
6. `.agents/skills/` の必要なSkillを参照して作業する。
7. まずMarkdown成果物を作る。
8. 内容レビュー後に、必要に応じてHTML、PDF、PPTX、PNG、infographic promptへ展開する。

## この依頼でやってよいこと

- 4日間版のカリキュラムを作る。
- 子供向けスライド原稿を作る。
- 親向け説明スライド原稿を作る。
- ワークシート原稿を作る。
- インフォグラフィック生成用prompt packetを作る。
- Codex用Skillを追加・改善する。
- 既存の旧構成記述を4日間版に統合する。

## この依頼でやってはいけないこと

- 子供に起業を押し付ける。
- 「会社員は危険」「雇われるのは不安定」などと恐怖で動かす。
- 稼げること、勝つこと、順位を中心にする。
- 親を先生、採点者、ゲームマスターにする。
- 既存アニメ、漫画、ゲーム、ロゴ、特定作家の絵柄を模倣する。
- `git add .` を使う。
- `.env`、Cookie、Login Data、Cache、Crashpad dump、ブラウザプロファイルを含める。

## 固定した4日間テーマ

1. Day1: Value Detective / 仕事・お金・価値の正体を見つける
2. Day2: Problem Hunter / 日常から価値の種を見つける
3. Day3: Idea & Want Signal Builder / 誰の何を助けるかを形にし、小さく試す
4. Day4: Pitch Challenger / 価値の変化を3分で伝える

Day3は、以前分かれていた「アイデア作り」と「欲しいサイン・小さな実験」を統合する。
