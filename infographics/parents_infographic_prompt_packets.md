# Parent Infographic Prompt Packets

## 共通指定

- 16:9 warm facilitator guide infographic.
- Calm, trustworthy, parent-friendly.
- Less game-like than kids materials.
- Use gentle diagrams, question cards, OK/NG response sheets.
- Parent is not teacher, not judge, not game master.
- Japanese text should be added later in Markdown or slide layout.
- No fear-based career imagery.

## 共通レビュー観点

- 親が先生、採点者、ゲームマスターに見えないか。
- 子供の成果を採点する絵になっていないか。
- 不安で行動させる雰囲気になっていないか。
- 1ページ1メッセージになっているか。
- 文字を画像生成に任せすぎていないか。
- 既存作品、既存キャラクター、特定絵柄をまねていないか。

## 保存先

候補画像を作る場合は、必ず `assets/generated/parents/page_XX/candidates/` に置く。
人間レビュー前に `final/` へ置かない。

