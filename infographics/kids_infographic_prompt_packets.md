# Kids Infographic Prompt Packets

## 共通指定

- 16:9 landscape educational infographic.
- Warm, soft, friendly, organized.
- Middle-school friendly but not childish.
- Use original characters only.
- Use visual metaphor: cards, detective notes, before/after arrows.
- Keep text minimal. Japanese text will be added later.
- Do not imitate existing manga, anime, game, or brand styles.

## 共通レビュー観点

- 1ページ1メッセージになっているか。
- キャラクターが行動しているか。
- 価値がBefore/Afterで見えるか。
- 文字を画像生成に任せすぎていないか。
- 既存作品、既存キャラクター、特定絵柄をまねていないか。
- 競争、ランキング、点数化に見えないか。

## 保存先

候補画像を作る場合は、必ず `assets/generated/dayX/page_XX/candidates/` に置く。
人間レビュー前に `final/` へ置かない。

## Day1: Value Detective / 6 pages

### Page 1: 今日のミッション

Purpose: 仕事・お金・価値の正体を探る導入。

Prompt:

```text
Create a 16:9 warm educational infographic for Japanese middle school students. Scene: an original middle-school character named Yuu, the value detective, opens a detective notebook titled "Value Detective". Around the notebook are three clue cards with simple icons: work, money, value. Soft cream background, gentle navy headings, coral accents, friendly flat illustration. The mood is curious and inviting, not childish. Leave large clean space for later Japanese title text. No existing characters, no famous anime style, no logos, no dense text.
```

Later text:

```text
Day1 価値探偵
お金と仕事の正体を見破れ
```

### Page 2: 仕事は相手の変化

Prompt:

```text
Create a 16:9 warm infographic showing a clear before/after transformation. Left side: a student looks confused and tired with homework or test notes. Right side: the same student looks relieved and ready, with a small "I can start" feeling. In the center, a gentle arrow made of value cards. Original characters only, soft educational flat illustration, warm cream and pale blue background, plenty of space for Japanese text. No mocking expression, no exaggerated success, no dense text.
```

Later text:
