# Parent Gamified v2 Infographic Prompt Packets

## Purpose

子供向けDay1 v2画像の温かいカード・ノート調を使いながら、親への説明資料を作るためのprompt packetである。

親向けなので、画面は落ち着かせる。
親を先生、採点者、ゲームマスターにしない。
親は、問いを出す伴走者として見せる。

## Common Direction

Use these prompts for parent-facing explanation images that match the established Day1 v2 card-board style.

Visual style:

- 16:9 landscape educational infographic.
- Warm cream paper background, light wooden table, notebooks, clipped cards, sticky notes, pencils, soft shadows.
- Soft navy title labels, coral and green accents, warm yellow hint icons.
- Original Japanese family learning illustration.
- Gentle cel-shaded and watercolor texture mix.
- Calm, trustworthy, and friendly.
- Same visual world as the kids' Day1 card images, but less busy and less game-like.
- Parent and students sit side by side or across the table.
- Parent listens, points to question cards, or writes small notes.
- Japanese text should be added later in layout. Do not rely on generated text.

Negative prompt for every image:

```text
No existing anime, manga, game, movie, logo, brand, mascot, or character. No imitation of a specific artist or franchise style. No recognizable character silhouette, outfit, symbol, creature, logo, or power effect from an existing work. No ranking, score, trophy, battle victory, money worship, fear-based career imagery, judge table, red pen grading, teacher podium, parent as game master, parent controlling the child, or mocking of people who are struggling. No dense text.
```

Save candidate images under:

```text
assets/generated/gamified_v2/parents/page_XX/candidates/
```

Do not place images in `final/` before human review.

## Parent Page 1: この取り組みの目的

Purpose: 親が、このワークを「起業を押し付ける授業」ではないと理解する。

Visual message: 会社に入っても、自分で何かを始めても役立つ力を育てる。

Prompt:

```text
Create a 16:9 parent-facing educational infographic matching a warm cream paper and card-board style. Scene: a parent and two middle-school students sit at a light wooden table with notebooks and simple value cards. The parent is relaxed and listening, not lecturing. On the table, three soft cards show icons only: finding value, trying small, learning from reaction. Use navy title label, coral and green accents, warm yellow hint icon, gentle original Japanese illustration, no readable text required, large clean space for later Japanese title.
```

Later text:

```text
これは起業を押し付ける授業ではありません
誰かにとって価値あることを見つける練習です
```

Review checklist:

- 親が先生のように一方的に話していないか。
- 不安で行動させる雰囲気になっていないか。
- 「価値を見つける」「小さく試す」が見えるか。

## Parent Page 2: 親の役割は伴走者

Purpose: 親の関わり方を、採点ではなく問いとして見せる。

Visual message: 親は答えを教える人ではなく、問いを出す伴走者である。

Prompt:

```text
Create a 16:9 warm parent guide infographic in the same cream paper card style. Scene: a parent gently hands a question card to a student while the student writes in a notebook. The parent has no red pen, no grading sheet, no teacher podium. Nearby cards show question mark, magnifying glass, and small light bulb icons. Use calm navy labels and soft green accents, friendly original characters, clear empty area for later Japanese text.
```

Later text:

```text
親は答えを教える人ではありません
問いを出す伴走者です
```

Review checklist:

- 親が採点者に見えないか。
- 子供が自分で考えている場面になっているか。
- 問いカードが中心に見えるか。

## Parent Page 3: 4日間の見取り図

Purpose: 4日間の流れを親がひと目で理解する。

Visual message: 見つける、形にする、試す、伝える。

Prompt:

```text
Create a 16:9 parent-facing timeline infographic matching the Day1 v2 notebook and card-board style. A clean tabletop map shows four large cards arranged left to right with simple icons only: magnifying glass for value, glowing seed for value seeds, puzzle and small test tool for idea and experiment, mini stage for pitch. Parent and students look at the map together. Calm composition, warm cream background, navy labels, coral arrows, no dense text, no score or ranking.
```

Later text:

```text
4日間の流れ
見つける → 形にする → 試す → 伝える
```

Review checklist:

- 4日間が競争やステージクリアに見えないか。
- 流れが左から右にわかるか。
- 親が管理者ではなく共有者に見えるか。

## Parent Page 4: 価値は相手の変化

Purpose: 親が、価値を「すごい案」ではなく「相手の変化」として見られるようにする。

Visual message: 困った、面倒、退屈、わからないが少し良くなる。

Prompt:

```text
Create a 16:9 educational infographic in a warm card-table style. On a wooden table, before/after cards show gentle changes: confused to can-start, messy to organized, bored to interested, worried to relieved. A parent points to the change arrow while students compare cards. The before side is respectful and not mocking. The after side is small improvement, not huge success. Use cream paper, navy labels, coral arrows, green relief accents, original characters.
```

Later text:

```text
価値とは、相手の状態が少し良くなること
困った → 助かった
面倒 → ラク
```

Review checklist:

- Beforeの人を笑いものにしていないか。
- Afterが大げさな成功になっていないか。
- 価値が人の変化として見えるか。

## Parent Page 5: お金はありがとうのサイン

Purpose: お金を、勝ちや偉さではなく、価値が届いた対価として見せる。

Visual message: 価値が届くと、ありがとうが返ってくる。

Prompt:

```text
Create a 16:9 calm parent explanation infographic in the same warm notebook-card style. A simple flow on the table shows a student helping another student with a small checklist, the helped student smiles with relief, and a gentle thank-you token returns. Use icons like heart note, small coin token, and smile, but avoid luxury money or greed. Parent watches quietly from the side with a supportive expression. Warm cream, navy, coral, green, no dense text.
```

Later text:

```text
お金は、価値が届いたことへの対価
まず見るのは「誰がどう助かるか」
```

Review checklist:

- お金が主役になりすぎていないか。
- 稼げる人が偉いという印象がないか。
- ありがとうの流れに見えるか。

## Parent Page 6: 使いたい問い

Purpose: 親がそのまま使える問いを持てるようにする。

Visual message: 親は短い問いで、子供の観察を広げる。

Prompt:

```text
Create a 16:9 facilitator question-card infographic in a warm cream tabletop style. Five blank question cards are arranged neatly around a notebook, each with simple icons only: person, current workaround, happy change, small test, red signal route change. A parent chooses one card while two students discuss. The scene is calm and supportive, no answer key, no grading, no teacher posture. Leave large space for later Japanese question text.
```

Later text:

```text
誰が困っている？
今はどうしている？
それが変わると何がうれしい？
小さく試すなら何から？
```

Review checklist:

- 問いが親の命令に見えないか。
- 子供が話しやすい雰囲気か。
- 正解カードや採点表に見えないか。

## Parent Page 7: 避けたい声かけ

Purpose: 親が、先回りした否定や評価を避けられるようにする。

Visual message: 止める言葉より、考えが進む問いに変える。

Prompt:

```text
Create a 16:9 parent guide infographic with two soft tabletop zones. Left zone: muted stop cards with icons for red pen, crossed arms, money-only view, and rushed answer. Right zone: warm question cards with icons for listening, asking, trying small, and changing route. The parent moves one card from the stop zone toward the question zone. Gentle, not scolding, no harsh red warning, no dense text, same cream paper and navy label style.
```

Later text:

```text
避けたい声かけ
試す前に否定する
親が先に答えを決める
お金だけで見る
```

Review checklist:

- 親を責める絵になっていないか。
- NG表現が強く怖く見えないか。
- 「言い換えればよい」と見えるか。

## Parent Page 8: セミナー本番へのつなぎ

Purpose: 親が、本番で子供に何を見てほしいかを理解する。

Visual message: 本番では、誰の何をどう変えるかを見る。

Prompt:

```text
Create a 16:9 hopeful parent-facing infographic in the same warm card and notebook style. Two students walk toward a small seminar room entrance carrying value detective notebooks and cards. A parent stands slightly behind with a calm supportive expression, not leading or controlling. Floating card icons show specific person, before/after change, small test, and pitch. Warm cream background, navy label, coral and green accents, no fear imagery, no pressure.
```

Later text:

```text
セミナー本番では
誰の何をどう変えるかを見てみよう
```

Review checklist:

- 本番が楽しみに見えるか。
- 親が主役になっていないか。
- 不安を煽っていないか。
