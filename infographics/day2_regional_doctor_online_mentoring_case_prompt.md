# Day2 Case Infographic Prompt: Regional Doctor Online Mentoring

## Purpose

Day2の「具体的な誰かを見る」ための見本画像prompt。

子供たちに、身近ではない医療の事業でも、次の形で見れば価値の種を見つけられることを示す。

```text
誰が、どんな状態で困っているか。
今はどうしているか。
それがどう変わるとうれしいか。
```

## Child-Friendly Case Summary

対象は、医師が少ない地域で働く若手医師。

その人は、近くに相談できる指導医が少なく、診療で迷ったときにすぐ聞けない。
そのため、自分の診療がこれでよいのか不安になる。

オンラインで経験ある指導医に相談できたり、短いレクチャーを受けられたりすると、不安が少し減る。
診療の考え方を学び、次にどう動くかを考えやすくなる。

## One Message

```text
価値の種は、具体的な誰かの不安の中にある。
```

## On-Slide Text

```text
地域で働く若手医師の価値の種
相談できない不安 -> オンラインで相談できる安心
```

## Speaker Line

```text
これは、医師が少ない地域で働く若手医師の例です。
ポイントは「医療がすごい」という話ではなく、誰のどんな不安が少し良くなるかを見ることです。
```

## Child Action

- 「誰が困っているか」を言う。
- BeforeとAfterを見つける。
- 「次に聞く問い」を1つ考える。

## Visual Learning Check

| Item | Design |
|---|---|
| 画面タイプ | Case File + Evidence Board |
| 子供がすること | 具体的な誰か、Before、Afterを探す |
| キャラクターの行動 | ユウが若手医師のケースファイルを指し、ミホが問いカードを出す |
| 見せたい価値の変化 | 相談できず不安 -> オンラインで相談でき、次にどうするか考えやすい |
| ゲーム要素 | 価値の種カード、Before/After矢印、問いカード |
| 合格基準 | 子供が「地域で働く若手医師が、何に困っていて、どう助かるか」を言える |
