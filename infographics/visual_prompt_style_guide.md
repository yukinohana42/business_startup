# Visual Prompt Style Guide

## 共通スタイル

温かく、やわらかく、親しみやすい教育向けインフォグラフィック。

- 16:9 landscape。
- クリーム系または淡いブルー系背景。
- やわらかいネイビーの見出し。
- 落ち着いたコーラル、やさしいグリーン、あたたかいイエローをアクセントにする。
- フラットであたたかいイラスト。
- 中学生向けだが幼稚すぎない。
- 親が見ても安心できる。
- 余白を広めにする。
- 1ページ1メッセージ。

## キャラクター

完全オリジナルの同世代キャラクター。

- 価値探偵ユウ: ノートを持ち、観察する。
- アイデアビルダー・ソラ: カードやラフスケッチを並べる。
- ナビゲーター・ミホ: 問いを書いたカードを示す。

既存漫画、アニメ、ゲーム、映画、ブランドの絵柄やキャラクターを模倣しない。

## 図解要素

- Before/After。
- 矢印。
- カード。
- 探偵ノート。
- 実験メモ。
- ピッチステージ。
- 付箋。
- 進捗ゲージ。ただし評価ではなく学びの記録。

## 文字の扱い

画像生成に正確な日本語本文を任せない。

Promptには「短いラベルのみ」「日本語本文は後編集」と明記する。

## Negative Prompt

- no existing anime character
- no famous manga style
- no brand logos
- no aggressive game UI
- no neon cyberpunk style
- no ranking board
- no money worship
- no fear-based career imagery
- no dense text
- no tiny captions
