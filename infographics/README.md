# Infographics

スライド生成が難しい場合や、Zoomで概念を直感的に見せたい場合は、複数枚のインフォグラフィックを使う。

## 基本方針

- 1枚に詰め込まない。
- 1ページ1メッセージ。
- 4〜6枚で1つの流れを作る。
- 日本語本文は正確性を優先し、後編集で載せる。
- 画像生成は背景、人物、図解、カード、雰囲気を作るために使う。
- 完全オリジナルキャラクターを使う。
- 既存作品や特定絵柄を模倣しない。
