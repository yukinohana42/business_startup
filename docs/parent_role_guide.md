# 親の役割ガイド

## 親は何をする人か

親は、先生ではない。
親は、採点者ではない。
親は、ゲームマスターでもない。

親は、問いを出す伴走者である。

## 親が支えるもの

- 子供の観察。
- 子供の発見。
- 小さな挑戦。
- 反応から学ぶこと。
- 自分の言葉で説明すること。

## 親が先に正解を言わない理由

親が先に正解を言うと、子供は自分で観察するより「親が喜ぶ答え」を探しやすくなる。

このワークショップでは、答えのきれいさより、本人の観察と言葉を大切にする。

## 親のOK声かけ

| 場面 | OK声かけ |
|---|---|
| アイデアが出ない | 今日、自分がちょっと面倒だったことから探してみよう |
| 対象が広すぎる | それを一番使いそうな1人は誰かな？ |
| 解決案が大きすぎる | まず1日だけ試すなら、何ができる？ |
| 反応が悪い | 赤信号だね。どこを変えるサインだと思う？ |
| ピッチが長い | 一番伝えたい変化は何？ |

## 親のNG声かけ

| NG声かけ | なぜ避けるか |
|---|---|
| 試す前に否定する | 子供が試す前に止まる |
| 案を大きくするよう急かす | 身近な価値を軽く見てしまう |
| お金だけで見る | お金だけが目的に見える |
| 親が先に答えを決める | 子供が親の答えを探す |
| よくある案だと切り捨てる | 観察する意欲が下がる |

## 親が見たい変化

4日後、子供が次のように言えたら十分。

- これって誰が助かるんだろう？
- その人は今どうしてるんだろう？
- まず小さく試すなら何ができるかな？
- 赤信号なら変えればいいんだよね。
- 起業セミナーで、どんな価値を作っているか見てみたい。

## 親向けの注意

このワークショップは、将来の不安を背負わせる時間ではない。

働き方への不安で急かすのではなく、「会社に入っても、自分で何かを始めても、人に価値を届ける力は役立つ」と伝える。
