# Gamified Visual v2 Direction

## Purpose

v1のシンプルな教材は残す。
v2では、中学2年生男子が入り込みやすいように、スライドとカードをよりゲーム的で冒険感のある見た目にする。

ただし、学習目的は変えない。
目的は「誰かにとって価値あることを見つけ、形にし、小さく試す力」を育てること。

## IP and Style Rule

既存の漫画、アニメ、ゲームのキャラクター、ロゴ、固有の絵柄は使わない。
画像生成promptにも、既存作品名、既存キャラクター名、作者名を入れない。

使ってよいのは、次のような一般的な構造だけである。

- 仲間と冒険する。
- 小さな相棒キャラクターがいる。
- カードを集めて組み合わせる。
- モヤモヤを発見して変える。
- 修行のように少しずつ技を覚える。
- 赤信号を見て作戦を変える。

## World Concept

### Value Quest

子供たちは「価値クエスト」に参加する。
敵を倒す話ではなく、町や学校にある「モヤモヤ」を見つけて、誰かの状態を少し良くする冒険である。

### Core Metaphors

| 学習概念 | v2での見せ方 |
|---|---|
| 価値 | モヤモヤが晴れて、相手の表情や行動が変わること |
| 価値の種 | フィールドに落ちている小さな光の種 |
| 助けたい人 | ケースファイルに載る具体的な人 |
| 欲しいサイン | レーダーに反応する小さな光 |
| 小さな実験 | 試作ラボで作る小さなお試し道具 |
| 赤信号 | 進路を変えるゲート |
| ピッチ | 仲間に作戦を伝えるミニ発表ステージ |

## Original Characters

### ユウ: 価値探偵

- 役割: Before/Afterを見つける。
- 行動: 虫眼鏡、探偵ノート、変化カードを使う。
- 性格: 落ち着いて観察する。

### ソラ: アイデアビルダー

- 役割: カードを組み合わせてアイデアを形にする。
- 行動: ラフスケッチ、試作ラボ、サインレーダーを使う。
- 性格: まず試す。直すことを怖がらない。

### ミホ: クエストナビ

- 役割: 正解ではなく問いを出す。
- 行動: クエストボード、問いカード、振り返りログを使う。
- 性格: 伴走する。採点しない。

### モヤモヤ

- 役割: 困った、面倒、退屈、わからない、忘れそう、続かないなどを表す完全オリジナルの小さなキャラクター。
- 見せ方: かわいすぎず、怖すぎない。相手を笑いものにしない。
- 変化: Afterでは、モヤモヤが小さな光や道具に変わる。

## Slide Mood

- 16:9。
- 1画面1ミッション。
- 背景は「町」「学校」「部室」「試作ラボ」「小さなステージ」など。
- 文字は少なく、絵で行動がわかる。
- 少年向けの勢いはあるが、バトルや勝敗を中心にしない。
- 色はネイビー、コーラル、グリーン、イエロー、淡いブルーを使う。

## Card Mood

- カードは「集める」「選ぶ」「組み合わせる」道具にする。
- レア度、点数、ランキングは使わない。
- カード種別は色で分ける。

| Card Type | Visual |
|---|---|
| 価値変化カード | Before/Afterの矢印と光 |
| 価値の種カード | フィールドに落ちた小さな種 |
| 助けたい人カード | ケースファイル |
| 欲しいサインカード | レーダー反応 |
| ピッチ質問カード | クエストナビからの問い |

## Visual Gate

画像生成やPPTX化の前に確認する。

- 既存作品に見えない。
- キャラクターが行動している。
- 子供が見る、選ぶ、書く、話す、試す、直す、発表する行動につながる。
- 価値が人のBefore/Afterとして見える。
- 親が採点者やゲームマスターになっていない。
- 競争、ランキング、勝敗に見えない。
