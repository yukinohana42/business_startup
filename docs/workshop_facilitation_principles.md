# 進行原則

## 基本姿勢

このワークショップは、親が知識を教え込む時間ではない。

子供が自分の生活から「誰かが助かる変化」を見つけ、言葉にし、小さく試すための時間である。

## 進行役の合言葉

「説明を短く、問いを多く、答えを急がない。」

## 1回45分の基本配分

| 時間 | 内容 | 進行のコツ |
|---|---|---|
| 0〜5分 | 今日のミッション | ゲームの始まりのように短く伝える |
| 5〜12分 | 導入問い | 正解を求めない |
| 12〜20分 | 身近な例で説明 | 例は1〜2個で十分 |
| 20〜35分 | メインワーク | 親は黙って見守る時間を作る |
| 35〜42分 | 共有 | すごさではなく発見を聞く |
| 42〜45分 | まとめ | 次に見ること、聞くことを決める |

## 親が使う問い

- 誰が困っている？
- 今はどうしている？
- それが変わると何がうれしい？
- 使った後、その人はどう変わる？
- もっと小さく試すなら何ができる？
- 赤信号なら、どこを変える？

## 子供が止まった時の助け方

### NG

「ちゃんと考えて。」

### OK

「じゃあ、自分の今日1日で面倒だったことから探してみよう。」

### NG

「仕事にはならないよ」と決めつける。

### OK

「それで助かる人が1人いるとしたら、誰だろう？」

### NG

「もっと大きいアイデアにして。」

### OK

「まず小さく試すなら、どこからできるかな？」

## 褒めるポイント

アイデアの完成度ではなく、行動を褒める。

- よく観察したね。
- 具体的な人を考えられたね。
- BeforeとAfterが見えたね。
- 小さく試す方法を考えられたね。
- 赤信号を学びに変えられたね。

## 避ける進行

- 親が講義し続ける。
- 正解を当てさせる。
- 稼げそうかどうかだけを見る。
- 2人を比較する。
- 子供のアイデアを大人が作り替えすぎる。
- 起業セミナー本番で恥をかかないように、という不安で動かす。

## 実施後の記録

各日終了後、親は次を短く記録する。

- 子供が反応した例。
- 子供が難しそうだった言葉。
- 盛り上がったワーク。
- 次回減らす説明。
- 次回増やす問い。
