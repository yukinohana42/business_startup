# 中学生向け用語集

## 仕事

誰かの状態を少し良くすること。

例:

- お腹がすいた人に、おいしいご飯を出す。
- わからない人に、わかるように教える。
- 探すのが面倒な人に、すぐ見つかるようにする。

## 価値

相手の状態が変わること。

```text
困った → 助かった
面倒 → ラク
退屈 → 楽しい
わからない → できそう
欲しい → 手に入った
```

## お金

価値が届いたことへの対価。

「ありがとう」が形になったもの、と考えるとわかりやすい。

## 顧客

それを本当に欲しい具体的な人。

「みんな」ではなく、「誰が？」を考える。

## 問題

その人のBeforeの状態。

例:

- テスト前に、何から勉強すればいいかわからない。
- 部活の準備で忘れ物が多い。
- ゲーム攻略の情報を探すのに時間がかかる。

## 解決案

BeforeをAfterに変える方法。

## PMF

作ったものと、欲しい人の気持ちがピタッとはまること。

サイン:

- 欲しいと言う。
- 使ってみたいと言う。
- もう一回使う。
- 友達に教える。
- 自分から質問してくる。

## 実験

完璧に作る前に、小さく試して反応を見ること。

例:

- 紙に書いて見せる。
- 友達に1つ質問する。
- 1日だけ試す。
- チャットで聞く。
- 5分だけ使ってもらう。

## 赤信号

失敗ではない。

「次に変える場所が見えた」というサイン。

## ピッチ

自分たちのアイデアが、誰の何をどう変えるのかを短く伝えること。

すごさ自慢ではなく、価値の変化を伝える時間。
