# CODEX MASTER PROMPT

あなたはCodexです。このプロジェクトでは、中学2年生2人向けの4日間・起業セミナー事前ワークショップ教材を準備します。

2. `01_PROJECT_REQUIREMENTS.md`
3. `02_SOURCE_OF_TRUTH.md`
4. `03_FOUR_DAY_CURRICULUM.md`
5. `04_DELIVERABLES_SPEC.md`

7. `AGENTS.md`
8. `DESIGN.md`
9. 必要な `.agents/skills/*/SKILL.md`

## 目的

子供たちが、次を理解できる4日間の教材を作ってください。

- 価値とは、相手の状態が少し良くなること。
- 仕事とは、誰かへの価値提供であること。
- お金は、価値が届いたことへの対価であること。
- 価値の種は、困った、面倒、好き、欲しい気持ちの中にあること。
- 良いアイデアは、具体的な誰かが本当に欲しいものに近づけること。
- 完璧に作る前に、小さく試して反応を見ること。

## 重要な制約

- 全日程は4日間。
- 起業を押し付けない。
- 「会社員は危ない」と言わない。
- 親を先生、採点者、ゲームマスターにしない。
- 子供が自分ごととして楽しく取り組めるようにする。
- 既存作品、既存キャラクター、特定絵柄を模倣しない。
