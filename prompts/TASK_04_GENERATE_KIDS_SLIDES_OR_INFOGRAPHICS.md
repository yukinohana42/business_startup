# TASK 04: Generate Kids Slides or Infographic Prompt Packets

## Goal

レクチャー時に使う子供向けスライドを完成させる。スライド生成が難しい場合に備え、インフォグラフィックprompt packetも整える。

## Read First

- `03_FOUR_DAY_CURRICULUM.md`
- `DESIGN.md`
- `docs/visual_learning_system.md`
- `.agents/skills/kids-slide-writer/SKILL.md`
- `.agents/skills/infographic-prompt-designer/SKILL.md`
- `.agents/skills/visual-storytelling-designer/SKILL.md`

## Tasks

1. `slides/kids/day1_lecture_slides.md` を完成させる。
2. `slides/kids/day2_lecture_slides.md` を完成させる。
3. `slides/kids/day3_lecture_slides.md` を完成させる。
4. `slides/kids/day4_lecture_slides.md` を完成させる。
5. `infographics/kids_infographic_prompt_packets.md` を各日4〜6枚程度で整える。
6. `infographics/parents_infographic_prompt_packets.md` を親向け6〜8枚程度で整える。

## Requirements

- 各日8〜12枚。
- 1スライド1メッセージ。
- 身近な例を入れる。
- 問いを入れる。
- ワークに自然につなげる。
- インフォグラフィックは1枚に詰め込まない。
- 日本語本文は後編集前提でもよい。

## Acceptance Criteria

- 子供が楽しく参加できそう。
- 起業セミナー本番とつながる。
- 価値が相手の変化として見える。
- PMFが中学生向けに言い換えられている。
- 既存IP模倣がない。
