# Day3 Facilitator Script: Idea & Want Signal Builder

## Overview

今日の目的は、具体的な誰かに向けてアイデアを作り、小さく試す方法を決めること。

PMFは「作ったものと欲しい人の気持ちがピタッとはまること」と言い換える。
子供の作業では「欲しいサイン」として扱う。

## Script

### 0:00-0:06 Opening

Parent says:

「今日はアイデア・欲しいサインビルダーです。誰の何を助けるかを形にします。」

Ask:

「前回選んだ価値の種は何だった？」

If stuck:

「困った、面倒、好き、欲しいの中から1つ思い出そう。」

### 0:06-0:14 Concept

Parent says:

「今日は『みんな』ではなく、具体的な1人を考えます。」

Ask:

「そのアイデアを本当に使いたい人は誰？」

If stuck:

「テスト前に迷う中2、朝練前に忘れ物が多い部活生、みたいに場面まで入れてみよう。」

### 0:14-0:28 Idea Sentence

Parent says:

「アイデアは、誰の何をどう変えるかで考えます。」

Ask:

「その人のBeforeは何？ Afterはどうなるとうれしい？」

If stuck:

「困ったを助かったにする？ 面倒をラクにする？ わからないをできそうにする？」

### 0:28-0:38 Small Experiment

Parent says:

「最初から完成品を作らなくてOKです。小さく試します。」

Ask:

「一番小さく試すなら何ができる？」

If stuck:

「紙に書いて見せる。友達に1つ聞く。1日だけ試す。5分だけ使ってもらう。どれが近い？」

### 0:38-0:43 Red Signal

Parent says:

「反応が薄くても、失敗ではありません。赤信号です。」

Ask:

「反応が薄かったら、どこを変える？」

If stuck:

「助けたい人、困りごと、言い方、試し方のどれを変えられそう？」

### 0:43-0:45 Final Mission

Parent says:

「最後に、今日のアイデアを一文にします。」

Ask:

「私たちは、誰の何を何に変える？」

If stuck:

「ワークシートのBeforeとAfterをそのまま使ってOK。」

## 30min Version

- 解決案は1つだけでよい。
- 欲しいサインは2つだけ選ぶ。
- 実験は1つだけ決める。

## 45min Version

- Time Plan通りに進める。
- 解決案は3つ出しても、実験は1つにしぼる。
- 赤信号で変える場所は1つだけ決める。

## 60min Version

- 助けたい人カードを使う。
- 解決案を3つ出し、いちばん小さい案を選ぶ。
- 赤信号なら変える場所を2つ考える。

## Do Not Say

- すぐに完成度を求める言い方。
- 使う人を「みんな」で済ませる言い方。
- 反応が悪いことを責める言い方。
- お金だけで案を見る言い方。
