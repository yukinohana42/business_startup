# Day1 Facilitator Script: Value Detective

## Overview

今日の目的は、仕事を「お金をもらうこと」だけでなく、「誰かの状態を少し良くすること」として見られるようにすること。

親は答えを教えない。
短い問いを出し、子供の言葉を拾う。

## Materials

- `slides/kids/day1_lecture_slides.md`
- `worksheets/day1_value_detective_worksheet.md`
- 必要なら `cards/value_transformation_cards.md`

## Time Plan

| Time | Slide/Work | Parent Says | Child Action |
|---|---|---|---|
| 0:00-0:05 | Slide 1 | 今日のミッションを伝える | 今日やることを聞く |
| 0:05-0:12 | Slides 2-3 | 仕事のイメージを聞く | 自分の考えを話す |
| 0:12-0:20 | Slides 4-6 | 価値とお金をBefore/Afterで見る | 例を見て考える |
| 0:20-0:32 | Worksheet 2-6 | 身近な仕事を1つ選ぶ | 書く |
| 0:32-0:40 | Slides 7-8 | 誰がどう変わったかを聞く | 共有する |
| 0:40-0:45 | Slides 9-10 | Final Missionと観察を決める | 一文を書く |

## Script

### 0:00-0:05 Opening

Parent says:

「今日は価値探偵です。仕事やお金の正体を、探偵みたいに見てみます。」

Ask:

「仕事って、何をしたら仕事になると思う？」

If stuck:

「会社とか大人の話じゃなくてOK。ラーメン屋、コンビニ、塾、動画でも考えてみよう。」

### 0:05-0:12 Opening Question

Parent says:

「今の考えで大丈夫です。正解を当てる時間ではありません。」

Ask:

「お金をもらっていなくても、誰かが助かっていたら仕事っぽいと思う？」

If stuck:

「家で誰かを手伝った時、相手はどう変わった？」

### 0:12-0:20 Concept

Parent says:

「今日の見方は、仕事とは誰かの状態を少し良くすること、です。」

Ask:

「ラーメン屋さんなら、食べる前と後で人はどう変わる？」

If stuck:

「お腹がすいた、から、どうなる？」

### 0:20-0:32 Main Work

Parent says:

「身近な仕事を1つ選びます。すごい仕事でなくてOKです。」

Ask:

「その仕事で助かる人は誰？」

If stuck:

「学校帰りの中学生、急いでいる人、わからなくて困っている人、みたいに1人を思い浮かべよう。」

### 0:32-0:40 Share

Parent says:

「発表では、すごいかどうかではなく、変化を見ます。」

Ask:

「Beforeは何？ Afterは何？」

If stuck:

「困った、面倒、退屈、わからない、欲しい、のどれに近い？」

### 0:40-0:45 Final Mission

Parent says:

「最後に、自分の言葉で一文を完成させます。」

Ask:

「仕事とは、何を何に変えること？」

If stuck:

「今日選んだ仕事のBeforeとAfterをそのまま入れてみよう。」

## 30min Version

- Slides 4-6はラーメン屋の例だけにする。
- ワークは1つの仕事だけにする。
- 共有は1人1分にする。

## 45min Version

- Time Plan通りに進める。
- ワークは身近な仕事を1つにしぼる。
- 共有はBefore/Afterだけにする。

## 60min Version

- 仕事を2つ選ぶ。
- 価値変換カードを使って、Before/Afterを並べる。
- 最後に「お金は何へのありがとうか」をもう一度話す。

## Do Not Say

- 試す前に否定する言い方。
- お金だけで見る言い方。
- 親が先に答えを決める言い方。
- 子供2人を比べる言い方。
