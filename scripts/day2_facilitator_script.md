# Day2 Facilitator Script: Problem Hunter

## Overview

今日の目的は、価値の種が日常の中にあると気づくこと。

価値の種は、困った、面倒、退屈、好き、欲しい、もっとやりたい気持ちの中にある。
親は観察を広げる。

- `slides/kids/day2_lecture_slides.md`

|---|---|---|---|
| 0:00-0:05 | Slide 1 | 今日のミッションを伝える | 前回の観察を思い出す |
| 0:05-0:12 | Slides 2-4 | 価値の種を説明する | 面倒だったことを出す |

## Script

### 0:00-0:05 Opening

Parent says:

「今日は困りごとハンターです。日常に隠れている価値の種を探します。」

Ask:

「前回見つけた、誰かが助かっている仕事は何だった？」

If stuck:

「家、学校、街、スマホの中で見たものを思い出そう。」

### 0:05-0:12 Concept

Parent says:

「価値の種は、大発明だけではありません。小さな面倒や、もっと楽しくしたい気持ちにもあります。」

Ask:

「今日、ちょっと面倒だったことは何？」

If stuck:

「朝、学校、部活、スマホ、友達との予定、どこかで探してみよう。」

### 0:12-0:25 Main Work

Parent says:

「まず見る場所を選びます。全部の欄をうめなくてOKです。」

Ask:

「自分の1日、家、学校、部活、スマホ。どこが一番見つけやすそう？」

If stuck:

「自分が少しイライラしたことから始めよう。」

### 0:25-0:35 Deep Dive

Parent says:

「1つだけ選んで、誰の種なのかを見ます。」

Ask:

「それで一番困っている、または欲しがっている人は誰？」

If stuck:

「自分、友達、家族、部活の人、先生、お店の人の中で近い人はいる？」

### 0:35-0:42 Share

Parent says:

「困りごとだけでなく、好きやもっとやりたいも価値の種です。」

Ask:

「それが変わると、その人は何がうれしい？」

If stuck:

「ラクになる？ 楽しくなる？ 早くなる？ 続けやすくなる？」

### 0:42-0:45 Final Mission

Parent says:

「最後に、次に聞いてみたい質問を1つ決めます。」

Ask:

「相手に聞くなら、何を聞く？」

If stuck:

「『今どうしてる？』『何が面倒？』『どうなったら楽？』から選んでOK。」

## 30min Version

- 観察場所は1つだけ選ぶ。
- 価値の種メモは3行だけ書く。
- 質問は1つだけ決める。

## 45min Version

- Time Plan通りに進める。
- 観察場所は1つか2つにする。
- 共有は「誰がどう助かるか」だけにしぼる。

## 60min Version

- 価値の種カードを使う。
- 2つの種を比べる。
- 1つ選んだ理由を話す。

## Do Not Say

- 困っている人を笑う言い方。
- すぐに案を大きくする言い方。
- 親が先に価値の種を決める言い方。
- 子供2人の発見を比べる言い方。
