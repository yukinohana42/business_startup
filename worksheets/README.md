# Worksheets

このフォルダには、4日間の子供向けワークシートを置く。

## 方針

- 完成度の高い事業計画書を作るためではない。
- 子供が考えを見える形にするためのメモである。
- A4印刷、またはZoom中の手元メモとして使える。
- 書く量は少なめにする。
- 「わからない」と書いても進められる。
- 最後に必ず次の観察、質問、試すことを入れる。
- 全部の欄を完璧に埋めなくてよい。
- 1つの欄は、短い言葉で書ければよい。

## 4日間のシート

1. `day1_value_detective_worksheet.md`: 価値変換ワーク。
2. `day2_problem_hunter_worksheet.md`: 価値の種ハンティング。
3. `day3_idea_pmf_experiment_worksheet.md`: アイデア文、欲しいサイン、小さな実験。
4. `day4_pitch_reflection_worksheet.md`: 3分ピッチと振り返り。
5. `all_days_workbook.md`: 4日分を1冊にまとめた版。
