# Skill: card-designer

## Purpose

印刷用または画面共有用のカード教材を作るSkill。

## Use When

- 価値カード、問題カード、顧客カード、PMF信号カード、ピッチ問いカードを作る時。
- 子供が選び、並べ、組み合わせるワークを作る時。

## Required Inputs

- `cards/card_system_spec.md`
- 対象Dayのワーク内容
- `DESIGN.md`

## Card Rules

- 1カード1メッセージ。
- 文字は短く。
- A4印刷で切りやすい。
- 色は意味ごとに分ける。
- カードは説明の飾りではなく、子供の行動を進める道具。

## Process

1. 対象ワークで必要な思考ステップを分解する。
2. 各ステップをカード化する。
3. カード文言を中学生向けに短くする。
4. カードの使い方を説明する。
5. A4印刷レイアウト案を作る。
6. 競争やランキングにつながらないか確認する。

## Output Format

```markdown
# Card Set: Name

## Purpose
## How to Use
## Cards
| ID | Front | Back/Prompt | Use Day |
```

## Acceptance Criteria

- 子供が選んで使える。
- どのワークで使うか明確。
- 言葉が短い。
- A4化しやすい。

## Must Not

- カードを能力評価に使わない。
- 点数カードにしない。
- 稼げるかランキングにしない。
