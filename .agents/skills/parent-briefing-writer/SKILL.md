# Skill: parent-briefing-writer

## Purpose

親向け説明資料、親向けスライド、親の関わり方ガイドを作るSkill。

## Use When

- `slides/parents/parent_explanation_slides.md` を作る時。
- 親向けFacilitator Guideを作る時。
- OK/NG声かけ表を作る時。

## Required Inputs

- `02_SOURCE_OF_TRUTH.md`
- `docs/parent_role_guide.md`
- `03_FOUR_DAY_CURRICULUM.md`

## Tone

- 落ち着いた説明。
- 教育的で安心感がある。
- 子供を急かさない。
- 親を責めない。
- 不安を煽らない。

## Process

1. この取り組みが起業の押し付けではないことを最初に明示する。
2. 育てたい力を説明する。
3. 価値、仕事、お金を親向けに整理する。
4. 4日間の流れを説明する。
5. 親の役割を「問いを出す伴走者」として説明する。
6. OK/NG声かけを具体的に入れる。
7. 成果物の完成度ではなく、子供の変化を見ることを伝える。

## Output Format

Markdown slide形式またはGuide形式。

```markdown
# Parent Briefing Slides

---

# Slide Title

Short parent-facing message.
```

## Acceptance Criteria

- 親が安心して見られる。
- 起業を強制しない。
- 親が先生や採点者にならない。
- 具体的な声かけがある。
- 4日間の意味が伝わる。

## Must Not

- 「会社員は危険」と煽らない。
- 親が子供を管理する資料にしない。
- 子供の成果を採点する表にしない。
