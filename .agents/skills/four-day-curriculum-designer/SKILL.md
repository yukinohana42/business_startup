# Skill: four-day-curriculum-designer

## Purpose

4日間の家庭内ワークショップを、学習目標、導入問い、ワーク、共有、宿題まで含めて設計するSkill。

## Use When

- 4日間カリキュラムを作る時。
- 30分、45分、60分版の時間配分を作る時。
- 旧構成で分かれていたアイデア作りとPMFを統合する時。
- 各日の目的、問い、ワークを整える時。

## Required Inputs

- `02_SOURCE_OF_TRUTH.md`
- `03_FOUR_DAY_CURRICULUM.md`
- 対象者: 中学2年生2人
- 実施時間: 30〜60分、標準45分

## Curriculum Spine

1. Day1: 仕事・お金・価値。
2. Day2: 価値の種の見つけ方。
3. Day3: アイデア、PMF、小さな実験。
4. Day4: 3分ピッチと振り返り。

## Process

1. 各日の中心概念を1つに絞る。
2. 子供が最後に言える一文を決める。
3. 導入問いを作る。
4. 身近な例を2つ以上用意する。
5. メインワークを1つ決める。
6. 共有方法を決める。
7. 宿題または観察ミッションを決める。
8. 30分、45分、60分版に展開する。

## Output Format

```markdown
# DayX: Title

## Theme
## Goal
## Child Can Say
## Key Concept
## Opening Question
## Everyday Examples
## Main Work
## Materials
## 30min Plan
## 45min Plan
## 60min Plan
## Parent Questions
## Final Mission
## Homework
```

## Acceptance Criteria

- 1日1テーマになっている。
- 子供の行動が含まれている。
- 親が話しすぎる構成になっていない。
- 具体例が中学生に近い。
- Day3にアイデアとPMFと小さな実験が統合されている。

## Must Not

- 事業計画書講座にしない。
- 専門用語の暗記にしない。
- 稼げるアイデア探しにしない。
