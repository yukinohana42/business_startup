# Skill: facilitator-script-writer

## Purpose

親が当日読める進行台本を作るSkill。

## Use When

- 各日の実施台本を作る時。
- スライドに合わせた話す内容を作る時。
- 親が話しすぎないように台本を整える時。

## Required Inputs

- 各日のスライド原稿。
- 各日のワークシート。
- `docs/workshop_facilitation_principles.md`
- `docs/parent_role_guide.md`

## Script Principles

- 親の説明は短くする。
- 1回の説明は60〜90秒以内を目安にする。
- 子供に問いを返す。
- 子供の答えをすぐ評価しない。
- 「誰がどう変わる？」に戻す。

## Process

1. 各スライドの目的を確認する。
2. 親が話す文を自然な口語で書く。
3. 子供に投げる問いを1つ以上入れる。
4. 子供が止まった時の助け舟を入れる。
5. NG声かけを避ける。
6. 時間配分を入れる。

## Output Format

```markdown
# DayX Facilitator Script

## Overview

## Time Plan
| Time | Slide/Work | Parent Says | Child Action |

## Script
### 0:00-0:05 Opening
Parent says:
"..."

Ask:
"..."

If stuck:
"..."

## Do Not Say
```

## Acceptance Criteria

- 親がそのまま読める。
- 子供が考える時間がある。
- すべての問いが短い。
- 説明が長くない。
- 親が採点者になっていない。

## Must Not

- 子供の答えを大人が作り替えすぎない。
- 起業セミナーで恥をかかないように、という不安で動かさない。
