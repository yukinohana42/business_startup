# Card System Spec

カードは必須成果物ではないが、子供が楽しく考えるための補助教材として推奨する。

## カードの目的

親が説明しすぎず、子供が選び、並べ、組み合わせて考える入口を作る。

## カード種類

### 1. 価値変換カード

Before/Afterを考えるためのカード。

例:

- 困った → 助かった
- 面倒 → ラク
- 退屈 → 楽しい
- わからない → できそう
- 欲しい → 手に入った

### 2. 価値の種カード

Day2で使う。

例:

- 困った
- 面倒
- 好き
- 欲しい
- もっとやりたい
- 退屈

### 3. 助けたい人カード

Day3で使う。

例:

- テスト前に迷う中2
- 朝練前に忘れ物が多い部活生
- 文化祭準備で忙しい係
- ゲーム攻略情報を探している人
- 家の手伝いを面倒に感じる人

### 4. 欲しいサインカード

Day3で使う。

例:

- 欲しいと言った
- 使ってみたいと言った
- もう一回使った
- 友達に教えた
- 自分から質問した
- 反応が薄かった

### 5. ピッチ問いカード

Day4で使う。

例:

- 誰が助かる？
- Beforeは何？
- Afterは何？
- 最初にどう試す？
- 赤信号ならどこを変える？

## デザインルール

- A4印刷して切れる配置。
- 1カード1メッセージ。
- 文字は大きく。
- 色は意味ごとに分ける。
- 勝敗やランキングに使わない。
- 学びの行動を支える道具として使う。
