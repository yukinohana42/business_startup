# Card Set: 価値の種カード

## Purpose

Day2で使う。

身近な中から「価値の種」を見つける。
価値の種とは、だれかのBefore/Afterにつながりそうな小さな気づきのこと。

## How to Use

1. 気になるカードを2枚まで選ぶ。
2. 「最近、自分や周りであったこと」に変える。
3. 助かる人がいるか考える。

まだアイデアになっていなくてもよい。
短い言葉で書く。

## A4 Layout

- A4に12枚を並べる。
- 表は一言だけにする。
- 裏は観察の問いにする。
