Day4 Facilitator Script: Pitch Challenger

Overview

今日の目的は、価値の変化を3分で伝えること。

ピッチはすごさ自慢ではない。 誰の何をどう変えるかを、短く自分の言葉で話す時間である。

Materials

  • slides/kids/day4_lecture_slides.md
  • worksheets/day4_pitch_reflection_worksheet.md
  • 必要なら cards/pitch_question_cards.md

Time Plan

Time Slide/Work Parent Says Child Action
0:00-0:05 Slide 1 今日のゴールを伝える アイデア文を確認する
0:05-0:12 Slides 2-4 ピッチの型を説明する 伝える順番を確認する
0:12-0:25 Worksheet 1-2 台本を作る 3分ピッチを書く
0:25-0:35 Slides 5-7 1回目を発表する 話す、聞く
0:35-0:42 Slide 8 / Worksheet 3-4 聞き手メモと改善をする 直す
0:42-0:45 Slides 9-10 ふり返りと本番接続をする 最後の一文を書く

Script

0:00-0:05 Opening

Parent says:

「今日はピッチチャレンジャーです。価値の変化を3分で伝えます。」

Ask:

「自分たちのアイデアは、誰の何をどう変えるものだった？」

If stuck:

「Day3のアイデア文をそのまま見よう。」

0:05-0:12 Pitch Frame

Parent says:

「ピッチはすごさ自慢ではありません。聞く人が『誰か助かりそう』と思えるように伝えます。」

Ask:

「聞く人が一番知りたいのは、何だと思う？」

If stuck:

「誰が助かる？ 何に困っている？ どう変わる？ の3つから考えよう。」

0:12-0:25 Write

Parent says:

「きれいな原稿でなくてOKです。短い言葉で書きます。」

Ask:

「まず最初の一文はどう言う？」

If stuck:

「私たちは、＿＿の＿＿を＿＿に変えるアイデアを考えました、で始めよう。」

0:25-0:35 First Pitch

Parent says:

「聞く人は採点しません。誰がどう変わるかをメモします。」

Ask:

「発表を聞いて、もっと聞きたいことは1つある？」

If stuck:

「誰が助かる？ Beforeは何？ Afterは何？ のどれかで聞こう。」

0:35-0:42 Improve

Parent says:

「1回話して終わりではありません。伝わったところを残して、わかりにくいところを短くします。」

Ask:

「一番伝わったところはどこ？」

If stuck:

「短くできる文を1つだけ選ぼう。」

0:42-0:45 Final Reflection

Parent says:

「最後に、起業セミナー本番で見るポイントを決めます。」

Ask:

「セミナーでは、どこに注目したい？」

If stuck:

「誰の困りごとを見つけたのか。どう小さく試したのか。どちらかを選ぼう。」

30min Version

  • ピッチは1分版にする。
  • 発表は1回だけにする。
  • 聞き手メモは「もっと聞きたいこと」だけ書く。

45min Version

  • Time Plan通りに進める。
  • 3分ピッチを1回行う。
  • 改善後は、直した一文だけをもう一度言う。

60min Version

  • 1回目と2回目のピッチを行う。
  • ピッチ問いカードを使う。
  • 2回目で短くなったところを確認する。

Do Not Say

  • 勝ち負けを決める言い方。
  • 点数をつける言い方。
  • 発表のうまさだけを褒める言い方。
  • セミナー本番への不安で急かす言い方。
