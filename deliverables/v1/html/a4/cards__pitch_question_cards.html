Card Set: ピッチ質問カード

Purpose

Day4で使う。

3分ピッチの前後に、聞き手がやさしく問いを返すためのカード。 問いは、アイデアを良くするために使う。

How to Use

  1. 聞き手はカードを1枚選ぶ。
  2. 発表者に短く質問する。
  3. 発表者は「今はわからない」でもよい。

質問は責めるためではない。 次の一歩を見つけるために使う。

A4 Layout

  • A4に12枚を並べる。
  • 表は質問を書く。
  • 裏は聞き方のヒントを書く。

Cards

ID Front Back / Prompt Use Day
PQ-01 だれが助かる？ 「みんな」ではなく、一人の場面で聞く。 Day4
PQ-02 何に困っている？ Beforeを短く聞く。 Day4
PQ-03 今はどうしている？ 今のやり方を聞く。 Day4
PQ-04 どう変わる？ Afterを聞く。 Day4
PQ-05 最初にどう試す？ 小さく試す方法を聞く。 Day4
PQ-06 赤信号ならどこを変える？ 人、場面、形、伝え方から選ぶ。 Day4
PQ-07 一番伝えたい変化は？ 1つにしぼって聞く。 Day4
PQ-08 もっと短く言うなら？ 15秒で言うなら何？ Day4
PQ-09 聞き手が知りたいことは？ まだ見えていないことを聞く。 Day4
PQ-10 セミナー本番で見ることは？ 本番で観察するサインを聞く。 Day4
PQ-11 何を変えたら使いやすい？ 直す場所を一つ聞く。 Day4
PQ-12 次に聞くならだれ？ もう一人、聞く相手を考える。 Day4

Facilitator Notes

  • 発表者を困らせる質問は避ける。
  • 聞き手には「なるほど」を先に言ってから質問してもらう。
  • 答えが出ない時は、ワークシートに「次に調べる」と書けばよい。
