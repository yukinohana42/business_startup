Card Set: 価値変化カード

Purpose

Day1で使う。

「価値」とは、相手の状態が少し良くなること。 このカードは、BeforeとAfterを見つけるために使う。

How to Use

  1. 気になるカードを1枚選ぶ。
  2. 「だれがそうなる？」を考える。
  3. 自分の身近な例に変える。

よい悪いを決めるためには使わない。 思いついたことを出すために使う。

A4 Layout

  • A4に12枚を並べる。
  • 1枚は名刺くらいの大きさにする。
  • 表は短い変化を書く。
  • 裏は問いを書く。

Facilitator Notes

  • 「すごい変化」でなくてよい。
  • 「少しラク」「少し安心」でも価値になる。
  • 子供が止まったら、学校、部活、家、友達、スマホ、テストの例に戻す。
